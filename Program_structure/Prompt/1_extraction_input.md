# 需求提取输入模板
*适用于Prompt1需求提取Agent*

## 输入格式

### 目标信息
**目标类型：** [网页URL / 软件名称 / 应用名称]
**目标描述：** [具体的网址链接或软件/应用的准确名称]

### 项目背景（可选）
**项目目标：** [你想开发什么类型的项目]
**目标用户：** [你的项目主要面向哪些用户]
**核心关注：** [你最关心目标软件的哪些方面]

---

## 填写说明

### 目标信息
- **目标类型**：明确指定是网页URL、软件名称还是应用名称
- **目标描述**：
  - 网页URL：提供完整的网址链接
  - 软件名称：提供准确的软件名称（如"微信"、"淘宝"、"12306"）
  - 应用名称：提供具体的应用名称和平台信息

### 项目背景
- **项目目标**：帮助Agent理解你的开发意图
- **目标用户**：帮助Agent分析用户需求和使用场景
- **核心关注**：帮助Agent重点分析你最感兴趣的功能模块

---

**目标用户：** 移动端购物用户

## 支持的目标类型

### 网页URL
- 电商网站（淘宝、京东、亚马逊等）
- 社交平台（微博、知乎、LinkedIn等）
- 工具网站（在线文档、项目管理等）
- 服务平台（订票、外卖、打车等）

### 软件名称
- 桌面应用（Office、Photoshop、VS Code等）
- 移动应用（微信、支付宝、抖音等）
- Web应用（Gmail、Slack、Notion等）
- 系统软件（操作系统组件、开发工具等）

### 应用名称
- iOS应用（App Store中的应用）
- Android应用（Google Play中的应用）
- 小程序（微信小程序、支付宝小程序等）
- Web应用（浏览器中运行的应用）

---

## 注意事项

1. **目标明确**：提供准确的目标信息，避免模糊描述
2. **合法合规**：确保分析的目标软件是公开可访问的
3. **目的正当**：用于学习和开发目的，不用于恶意复制
4. **范围适中**：选择功能复杂度适合你项目规模的目标
5. **信息完整**：尽可能提供完整的背景信息以获得更好的分析结果
